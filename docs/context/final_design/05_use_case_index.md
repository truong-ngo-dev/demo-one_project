# Use Case Index — Admin & Operator Portal (On-premise)

*Phiên bản sơ khai. Chi tiết flow sẽ bổ sung sau.*

---

## Admin Portal

| ID | Tên | Chức năng |
|---|---|---|
| UC-A1 | Quản lý người dùng hệ thống | Tạo / edit / deactivate tài khoản, assign role cho bất kỳ user |
| UC-A2 | Cấu hình tòa nhà | Thông tin building, cấu hình hệ thống, tích hợp hardware |
| UC-A3 | Quản lý roles & phân quyền | Xem / cấu hình PolicySet, UIElement visibility |
| UC-A4 | Audit log | Xem, filter, export log các action quan trọng |

---

## Operator Portal

| ID | Tên | Chức năng |
|---|---|---|
| UC-O1 | Quản lý cấu trúc vật lý | Tầng, unit (residential/commercial/common), tiện ích chung |
| UC-O2 | Onboard tenant doanh nghiệp | Tạo tenant org, gán space, tạo TENANT_ADMIN đầu tiên |
| UC-O3 | Onboard resident | Tạo tài khoản, gán unit, xác định quan hệ OWNER/LESSEE |
| UC-O4 | Quản lý BQL staff | Tạo tài khoản, assign role nội bộ (FINANCE/TECHNICAL/SECURITY/STAFF) |
| UC-O5 | Quản lý access card | Cấp / thu hồi thẻ, xử lý mất thẻ, approve request từ tenant |
| UC-O6 | Quản lý visitor | Pre-registered check-in, walk-in, blacklist |
| UC-O7 | Quản lý booking tiện ích | Xem lịch, approve/cancel, block facility |
